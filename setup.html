SETUP GUIDE

使用教程

Windows、Android、iOS / Mac 全平台客户端配置指南。订阅链接由你购买的机场提供，按对应平台步骤即可完成配置。

Platform 01

Windows 电脑

适用于 Windows 10 与 Windows 11。Windows 7 用户请联系机场客服获取旧版客户端。

安装与配置

01

下载并解压 v2rayN

从机场官网或客服处下载 v2rayN 的 Windows 安装包，将压缩文件解压到任意文件夹（建议新建一个固定目录长期使用）。

02

启动程序

打开解压后的文件夹，找到名为 v2rayN.exe 的文件（图标为字母 V），双击运行。

03

导入订阅链接

复制机场提供的订阅链接到剪贴板，然后：

  • 在 v2rayN 程序左上角点击 "配置"
  • 选择菜单第一项 "从剪贴板导入链接"
  • 订阅链接会自动导入并解析所有节点
04

设置系统代理

在程序最下方找到 "清除系统代理"，点击展开后选择 "自动配置系统代理"。

05

重启程序使配置生效

  • 在屏幕最右下角找到红色的 V 图标
  • 右键点击，选择 "退出"
  • 重新双击 v2rayN.exe 启动程序

常见问题

延迟显示 −1

正常运行时，软件右下角延迟应为正数。如显示 −1，通常是系统时间不准确导致。

解决方法：进入 Windows 系统设置 → 日期和时间 → 同步时钟，然后重启 v2rayN 即可。

使用 WhatsApp 桌面版

如需在 Windows 上使用 WhatsApp 客户端版本，请在软件最底部勾选 "启用 TUN" 选项，部分非浏览器应用才能走代理。

Platform 02

Android 手机

适用于所有 Android 系统手机与平板。推荐使用 v2rayNG 或 NekoBox。

安装与配置

01

安装 App

在 Android 手机上安装以下任意一个客户端（机场客服或官网通常会提供 APK 下载）：

  • v2rayNG — 经典款，功能完整
  • NekoBox — 界面更现代，支持协议更多
02

导入订阅链接

复制机场提供的订阅链接到剪贴板，然后：

  • 打开安装好的 App
  • 点击右上角的 "+" 图标
  • 选择菜单中的 "从剪贴板导入链接"

如果机场提供的是二维码，也可以在 App 内选择 "扫描二维码" 直接导入。

03

启动 VPN

  • 在 App 内选中导入的线路（点选高亮）
  • 点击 App 最底部的 三角形开关 启动连接
  • 系统弹出 VPN 权限请求时选择"允许"
Platform 03

iPhone · iPad · Mac

适用于所有苹果生态设备，推荐统一使用 Shadowrocket（小火箭）。

安装与配置

01

下载 Shadowrocket

使用 美区 Apple ID 登录 App Store，搜索并下载 Shadowrocket。该应用国区下架，必须使用海外区账号。

没有美区 Apple ID？可在 首页 · 美区 Apple ID 分类购买。

02

导入配置

  • 打开 Shadowrocket，点击左上角的 扫码按钮
  • 扫描机场提供的订阅二维码
  • 弹出任何权限请求，全部选择"允许"

如果只有订阅链接没有二维码，也可以点击右上角 "+" → 选择 "Subscribe" 粘贴链接导入。

03

启动 VPN

  • 导入成功后，在 App 内选中线路
  • 打开右上角的 VPN 开关 即可连接
首次连接提示

首次连接时，系统会弹出 "是否允许添加 VPN 配置" 的提示，请选择 允许。此后再次启动 VPN 不会重复弹出此提示。
